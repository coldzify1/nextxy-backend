
# Movie API

This project uses external api for movie data

## API Document
[API Document](https://nextxy-backend-production.up.railway.app/api)


## Project setup

```bash
$ yarn install
```

## Environment setup

Copy .env.example to .env


## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
$ yarn run test:cov
```
